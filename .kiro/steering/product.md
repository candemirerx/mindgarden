# Not Bahçesi - Product Overview

Not Bahçesi ("Note Garden") is a visual mind-mapping note-taking application with a tree and garden theme. Users organize their ideas as "trees" within "gardens" on an infinite canvas.

## Vision
"Plant your ideas in the soil, let them grow into trees."

## Core Concepts
- **Garden (Bahçe)**: A collection/workspace containing multiple trees
- **Tree/Node (Ağaç)**: Hierarchical notes with parent-child relationships
- **Canvas**: Infinite pan & zoom workspace for visual organization

## Key Features
- Google OAuth and email/password authentication via Supabase
- Infinite canvas with pan, zoom, and view state persistence
- Hierarchical tree nodes with expand/collapse
- Rich text editor with AI-powered Turkish spell checking (Gemini)
- PDF and Word export
- Data import/export (JSON)
- Android mobile app via Capacitor

## Target Users
Turkish-speaking users who prefer visual, hierarchical note organization.

## Language
- UI and content are in Turkish
- Code comments may be in Turkish or English
